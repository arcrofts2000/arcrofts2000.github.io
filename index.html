Portfolio

Here is a collection of projects I have worked on, both within University and as a Hobby.

Drop Fox

Created as part of the Wowie Jam 2.0 using Unity 5. Consists of simple tilt controls on Android and Cursor control using PC. Programmed in C#

ASCII Invaders

Created in my first year of Uni, in which we were given a proprietary university game engine and told to create Space Invaders but sideways. Programmed in C++

Untitled Adidas Game

In my first year, we had to work as a group to make a basic top down game using Unreal Engine 4 along with a Game Design Document. For this I took the role of Lead Game Design.

C++ Zombies Game

I was given the task of creating a top down zombie shooter using C++ and SFML.

Game Review Site

One module for the course consisted of Internet and Web Development. For this, I was tasked with creating a Dynamic Site using PHP and HTML.

ASCII Invaders: Remastered

This was my first project in my second year. This was created with C++ SFML. I decided to call it Remastered as it has a similar feel to the original ASCII Invaders I made in First Year, just with nicer graphics.

C++ Hieroglyph - Procedural Animation

I had a module for 3D Computer Graphics, in which I had to create a 3D World with a Car driving along a racetrack to show off different types of Transformations.

UE4 - AI Pathfinding

In this current semester, I need to make a couple of different levels in UE4 which consist of different types of AI Pathfinding - currently WIP.

UE4, Blueprint and C++ - Refactored Shooting Range

Currently I am working on refactoring an Unreal Engine Shooting Range project. This consists of two weapon types (projectile and ray) in two game modes, which I need to refactor into a single gamemode.

C++ SFML - Group Based

For this coursework, we are working in groups of 4 to create a 2D Top Down RPG. This time I am working as the Lead Player Developer, which consists of programming the Camera and Player, along with working alongside the Lead UI Dev to connect the Player information and the Lead Enemy / Pickup Dev to allow for Collisions.
